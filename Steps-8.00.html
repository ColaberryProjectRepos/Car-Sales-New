Import file into Power Bi Desktop 8

Dataset link:    https://www.kaggle.com/datasets/gagandeep16/car-sales

1.      Download the dataset from Kaggle and save it on your desktop

2.      Open Power BI Desktop

3.      Click on Get data button

4.      Select Text/CSV and click on Load

5.      A preview of the data will be displayed. Click on Load
