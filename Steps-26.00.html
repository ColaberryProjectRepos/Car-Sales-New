Line and Stacked Column Visual 1(Price by Month)

Steps

Click on a space on the canvas, and from the visualizations pane, select Line and Stacked Column Chart

Fill up the figures provided below for the necessary fields.

Build Visual Tab:

Shared axis = Date (Month)

Column Values = Price, Model

Line Values = YoY Price%, Sales YoY%, YoY FE%, YoY Hpower%, YoY ES%

Tooltips = Fuel Efficiency, Horsepower, Price, Sales, Engine Size (Liters), YoY Price%, Sales YoY%, YoY FE%, YoY Hpower%, YoY ES%,

After filling up the above rename fields, as per the gif provided below.

Sort the axis by Date, Month, and ascending order.

Format Visual Tab:

 

General: Properties, Size, Height=341, Width=529, Position, Horizontal=219, Vertical=100

                Title=On, Text=Price by Month, Font=Segoe UI (15), Text color=#FFFFFF, Background color=#0D6ABF, Horizontal alignment=Center

                Effects: Background=On, Background color=#E6E6E6, Transparency=30%,

 

Visual boarder=OFF, 

 

Visual: X-axis: Values: Font=Segoe UI (9), Font color=#000000

                          Title = Off

 

             Y-axis: Values: Font=Segoe UI (9), Font color=#000000, Display Units=Auto

                          Title = Off

 

             Gridlines: Horizontal=On, Style= Dotted, Color=#C8C6C4

 

             Lines: Colors: YoY ES% #6B2328, YoY Hpower%#6B007B, YoY FE%#09124F,Sales YoY%#E1C233,YoY Price%#118DFF

             Markers= Off

             Data labels=Off
