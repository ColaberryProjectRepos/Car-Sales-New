Create Dax Measure Table 14

1.    Select the Car_Sales Table and right click on it and select New Measure

2.    From the Power BI desktop ribbon, home tab, click on new measure

3.    The formula bar appears at the top of the report canvas.

4.    Fill in all the DAX formulas provided below, one at a time, and when you finish, Press Enter or select Commit (checkmark icon) in the formula bar.

 

5. After creating each measure choose the necessary format and decimal for each value (measure)

6. Save

 

DAX Formulas

. Average Price = AVERAGE(Car_sales[Price])

       . Average Horsepower = SUM(Car_sales[Horsepower])

       . Average Fuel Efficiency = SUM(Car_sales[Fuel_efficiency])

       .LY EF = CALCULATE([.Average Fuel Efficiency],DATEADD('Calendar'[Date],-1,YEAR))

       .LY ES = CALCULATE([.Total Engine Size],DATEADD('Calendar'[Date],-1,YEAR))

       .LY Price = CALCULATE([AVG Price],DATEADD('Calendar'[Date],-1,YEAR))

       .LY.Horsepower = CALCULATE([.Average Horsepower],DATEADD('Calendar'[Date],-1,YEAR))

       .Total Engine Size = SUM('Car_sales'[Engine Size (liters)])

       Total Car Sales = (( Sum(Car_sales[Price_in_thousands])*1000))

       .YoY Price%. = SUMX(VALUES('Calendar'[Year]),[.YoY_VAR])

YoY Engine Size% = SUMX(VALUES('Calendar'[Year]),[.YoY_VAR ES])

YoY Fuel Efficiency% = SUMX(VALUES('Calendar'[Year]),[.YoY_VAR FE])

YoY Hpower% = SUMX(VALUES('Calendar'[Year]),[.YoY_VAR Horsepower])

Sales YoY% =

DIVIDE(

    SUM('Car_sales'[Sales_in_thousands]) - CALCULATE(

        SUM('Car_sales'[Sales_in_thousands]),

        SAMEPERIODLASTYEAR('Calendar'[Date])

    ),

    CALCULATE(

        SUM('Car_sales'[Latest_Launch]),

        SAMEPERIODLASTYEAR('Calendar'[Date])

    )

) * 100

KPI IND YOY% = IF([Sales YoY%]<0,UNICHAR(9660),IF([Sales YoY%]>0,UNICHAR(9650),UNICHAR(UNICODE("-"))))

KPI IND YOY% price = IF([YoY% price]<0,UNICHAR(9660),IF([YoY% price]>0,UNICHAR(9650),UNICHAR(UNICODE("-"))))

KPI IND YOY% Horsepower = IF([.YoY% Horsepower Change]<0,UNICHAR(9660),IF([.YoY% Horsepower Change]>0,UNICHAR(9650),UNICHAR(UNICODE("-"))))

KPI IND YOY% Fule Efficiency = IF([YoY% Fuel_efficiency]<0,UNICHAR(9660),IF([YoY% Fuel_efficiency]>0,UNICHAR(9650),UNICHAR(UNICODE("-"))))

KPI IND YOY% = IF([Sales YoY%]<0,UNICHAR(9660),IF([Sales YoY%]>0,UNICHAR(9650),UNICHAR(UNICODE("-"))))

Waterfall Value = SWITCH (SELECTEDVALUE('Waterfall Chart KPI''s (2)'[Index]),1,'Car_sales'[.YOY Price%.],2,'Car_sales'[.Sales YOY%],3,'Car_sales'[YOY ES%],4,'Car_sales'[YOY FE%],5,'Car_sales'[YOY Hpower%])
