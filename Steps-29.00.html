Scatter Chart Visual (Price by Manufacture)

Steps
Click on the Scatter chart icon
from the Visuals pane to change the visual into a scatter chart 

Change figures provided below for the necessary fields

Build Visual Tab:

 

Values = Manufacture

X-Axis = Price

Y-Axis = YoY Price%

Size = Sales 

Tooltips = Fuel Efficiency, Horsepower, Price, Sales, Engine Size (Liters), YoY Price%, Sales YoY%, YoY FE%, YoY Hpower%, YoY ES%,

Format Visual Tab:

General: Properties, Size, Height=278, Width=529, Position, Horizontal=219, Vertical=442

Text: Price by Manufacture, Font, Segoe UI (15), Text Color#FFFFFF, Background Color#0D6ABF, Centerd Alignment

Tooltips: On, Font = Segoe UI (9), Label Color = #252423, Value Color #252423.Effect, Background=on,Color#E6E6E6,Transparency 30%,Visual border=Off

               

Visual: X-axis: Values: Font=Segoe UI (9), Font color=#00000 , Title = On

  Y-axis: Values: Font=Segoe UI  (9), Font color=#118DFF, Display Units=Auto,   Title = On

   Gridlines: Vertical=On, Color=#C8C6C4

  Markers: Colors= fx (click on the function tab and fill up values as per the image below)

    Color codes: Lowest Value=#41A4FF, Middle Value=#0D6ABF, Highest Value=#09124F
